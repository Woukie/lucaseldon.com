MealCoach AI

This is my latest project and my first with actual funding. MealCoach is an advanced food tracking app that leverages AI to record your food intake and analyze your diet against your personal goals. I'm particularly excited about this project as I'm soon to become a shareholder of the company.

Developing MealCoach went beyond just building an app and backend. Since this is a real-world product, it involved creating a whole ecosystem. The project consists of a landing page, administration tools for account management, hosting infrastructure, mail systems, security, and even required legal knowledge.

MealCoach is fully realeased on Apple and Android!

Communications Platform

Have you ever had to put "reddit" in a search query to get real results? Have you ever given up looking for something because every result is a low quality clickbait article?

The rise good SEO has ushered in the destruction of the usable web, and I have a solution.

We're building a federated communications platform that allows hosters to index threads in search engines. Our solution consists of the following stand-out features

  • Host your own server or let us host it for you
  • Complex custom behaviour support. Our in-server IDE lets server owners add custom behaviour to their server LIVE
  • We give server owners full control over the HTML of the site
  • Almost any communication style can be achieved through server customisation!
    • Create a group chat by only showing one thread and modifying the UI
    • Create a Discord-like environment by pining a few threads, labeling them as chat rooms and restricting sub-threads!
    • Some quite complex platforms can be made by allowing the modification of UI and adding scripts to control the behaviour of a server
    • The following configs will be provided for quick setup and learning material: DM, group chat, Discord, Twitter and an Image Board
  • Server owners can specify what is and isn't indexed. For example, server owners can only expose certain subthreads for search engine indexing, and can even programatically assign visibility of threads with our in-server JS API
